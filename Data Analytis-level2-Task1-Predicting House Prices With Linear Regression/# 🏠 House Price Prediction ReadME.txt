# 🏠 House Price Prediction using Linear Regression

A Machine Learning project that predicts house prices using the **Linear Regression** algorithm. This project demonstrates the complete machine learning workflow, including data preprocessing, exploratory data analysis (EDA), feature engineering, model training, evaluation, and visualization.

---

## 📌 Project Objective

The objective of this project is to build a regression model that predicts house prices based on property-related features such as location, square footage, BHK count, construction status, and resale information.

---

## 📂 Dataset

**Dataset Name:** `House_price_prediction.csv`

### Features

- POSTED_BY
- UNDER_CONSTRUCTION
- RERA
- BHK_NO.
- BHK_OR_RK
- SQUARE_FT
- READY_TO_MOVE
- RESALE
- ADDRESS
- LONGITUDE
- LATITUDE

**Target Variable**

- TARGET(PRICE_IN_LACS)

---

## 🚀 Project Workflow

- Import required libraries
- Load the dataset
- Perform Exploratory Data Analysis (EDA)
- Check missing values
- Analyze target variable distribution
- Encode categorical features using One-Hot Encoding
- Generate correlation heatmap
- Split data into training and testing sets (80:20)
- Train Linear Regression model
- Evaluate model using MSE, RMSE, and R² Score
- Visualize Actual vs Predicted values
- Perform Residual Analysis
- Analyze Feature Coefficients
- Compare Linear Regression with Ridge and Lasso Regression
